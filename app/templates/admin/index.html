The section of the app provides the ability to edit the following tables of the database:

  • Enrollment Term:
    • Please only edit the current term.
    • There must be one and only one tagged as the current term.
    • This must be updated every semester.
    • The list will update every night, there should be no need to add or remove any terms.
  • Grade Calculation:
    • Grade Rank: Ordered from Highest grade to Lowest
    • Grade: The letter grade
    • Threshold: Criteria 1 of the grade table
    • Min Score: Criteria 2 of the grade table

*Please exercise caution when you are editing any of these values. There is little data validation built in.
